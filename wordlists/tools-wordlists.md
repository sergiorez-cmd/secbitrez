## Criar Worlists Personalizadas Grep

Exemplo para criar uma wordlist de palavras com comprimento de 8 caracteres

$ grep -oE '\b\w{8}\b' /usr/share/wordlists/rockyou.txt > passwd-8lt.txt

$ wc -l passwd-8lt.txt
